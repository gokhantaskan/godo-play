# .impeccable.md

## Design Context

### Users

Casual gamers browsing for cross-platform multiplayer games. They arrive to discover new titles, check multiplayer support, and find games they can play with friends across platforms. Context is leisure — scanning covers, filtering by platform/mode, bookmarking games. Prioritize scannability, delight, and low-friction discovery.

### Brand Personality

**Bold, playful, vibrant.** The interface should feel like a curated gaming magazine — energetic but organized, colorful but not chaotic. Personality comes through in accent colors, micro-interactions, and confident typography. Never sterile, never cluttered.

### Aesthetic Direction

**Visual tone**: Dark, immersive, content-forward. Game artwork is the hero — the UI chrome recedes to let covers and metadata shine. Surfaces use subtle elevation (dark grays) rather than heavy borders.

**References**:

- **Letterboxd** — Content discovery UX, social logging patterns, dark theme with punchy accent colors, diary/list metaphors
- **Steam / GOG** — Rich game metadata, filtering power, community features, storefront browsing patterns
- **RAWG.io / IGDB** — Clean grid layouts, metadata-dense cards, database browsing UX

**Anti-references**: Generic dashboards, corporate SaaS, overly minimalist "empty" designs, neon-heavy gamer aesthetics (no RGB overload)

**Theme**: Dark mode — settled direction

- **Surface**: #15151a base (blue-purple tinted), layered containers (#1b1b21 → #373740)
- **Primary**: #b6a0ff (lavender) / #a98fff (container)
- **Secondary**: #4af8e3 (cyan/teal)
- **Semantic**: Green #55cc00, Yellow #ffc833, Red #e8573d
- **Text**: #cdd0da (body), #9da1b0 (muted), #e8e9f0 (headings/bright)
- **Borders**: rgba(200, 205, 230, 0.10)

**Typography**:

- Body: Manrope (300–700)
- Headings: Space Grotesk (300–700)
- Mono: Roboto Mono

### Design Principles

1. **Content is the hero** — Game artwork and metadata dominate; UI chrome stays minimal and recedes into dark surfaces
2. **Bold with purpose** — Lavender and cyan accents punch through the dark theme at intentional moments — CTAs, highlights, active states — never gratuitously
3. **Playful, not noisy** — Animations, transitions, and micro-interactions serve discoverability and delight, not decoration. Every motion earns its place
4. **Clarity at a glance** — Strong visual hierarchy for scanning: title, cover art, platforms, multiplayer mode. Users should grok a card in under 2 seconds
5. **Progressive detail** — Essentials upfront, depth on interaction. Cards → modals → full pages. Don't overwhelm the browse experience

### Accessibility

- **Target**: WCAG AAA compliance (7:1 contrast ratio for normal text, 4.5:1 for large text)
- **Motion**: Respect `prefers-reduced-motion` — all animations must have reduced/disabled alternatives
- **Color**: Never rely on color alone to convey meaning; pair with icons, text, or patterns
- **Keyboard**: Full keyboard navigation for all interactive elements
- **Screen readers**: Semantic HTML, ARIA labels where needed, meaningful alt text for game imagery
